# 🔐 Random Password Generator Project

This project contains two versions of a Password Generator built using Python.

---

## 🟢 1️⃣ Beginner Version (Command Line)

📂 File: beginner_password_generator.py

### Features:
- User-defined password length
- Choose letters, numbers, symbols
- Input validation
- Random password generation

### Concepts Used:
- random module
- string module
- User input handling

---

## 🔵 2️⃣ Advanced Version (GUI - Tkinter)

📂 File: advanced_password_generator.py

### Features:
- Graphical User Interface
- Password complexity options
- Strong password rules
- Clipboard copy feature
- Exclude specific characters

### Concepts Used:
- Tkinter
- Randomization
- GUI design
- Input validation
- Clipboard integration

---

## 🚀 How to Run

For Beginner Version:
python beginner_password_generator.py

For Advanced Version:
python advanced_password_generator.py
