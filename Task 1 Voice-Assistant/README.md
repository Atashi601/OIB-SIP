# 🎤 Voice Assistant Project

This project contains two versions of a Voice Assistant built using Python.

---

## 🟢 1️⃣ Beginner Version (Command Line)

📂 File: voice_assistant.py

### Features:
- Voice input using microphone
- Text-to-speech response
- Open Google
- Open YouTube
- Tell current time
- Exit command

### Concepts Used:
- speech_recognition
- pyttsx3
- datetime
- webbrowser
- Loops
- Exception handling

---

## 🔵 2️⃣ Advanced Version (GUI - Tkinter)

📂 File: advanced_voice_assistant.py

### Features:
- Graphical User Interface
- Button to start listening
- Display recognized text
- Open websites using voice
- Better interaction

### Concepts Used:
- Tkinter
- Speech Recognition
- Text-to-Speech
- Event Handling
- GUI Development

---

## 🚀 How to Run

Install required libraries:

pip install speechrecognition
pip install pyttsx3
pip install pyaudio

Run Beginner Version:
python voice_assistant.py

Run Advanced Version:
python advanced_voice_assistant.py
