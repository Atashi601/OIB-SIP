## 📝 Project Description
This project is a BMI (Body Mass Index) Calculator developed using Python.

It includes both:
- Beginner Level (Console-based BMI Calculator)
- Advanced Level (GUI-based BMI Calculator using Tkinter)

The application calculates BMI based on user input and displays the BMI category.

---

## 🚀 Features
- User input for weight and height
- Accurate BMI calculation
- BMI category display:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
- Input validation
- Error handling using try-except
- Simple and user-friendly GUI (Advanced Version)

- Tkinter (for GUI)
